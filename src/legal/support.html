Support - Office Syntax Highlighter

How to Use

Office Syntax Highlighter uses Shiki for syntax highlighting code in your PowerPoint presentations.

Note that this extension does not require any account, registration, or login.

Basic Operations

  1. Select your preferred language and theme, then either:
    • Replace the content of a selected text box, or
    • Create a new text box with your code
  2. To create a new text box with syntax highlighting, ensure no text is currently selected
  3. When creating a new text box, you can adjust its dimensions using the height and width controls below the text editor

Formatting Details

  • Tabs are automatically converted to 2 spaces while typing to optimize space usage in presentations
  • New text boxes are created using "Monaco" font by default, but you can modify this to match your preferences
